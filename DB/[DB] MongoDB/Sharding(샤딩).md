> # MongoDB Sharding
> **[ 직역 ] : Shard 는 전체 중 작은 부분을 뜻하며, sharding은 큰 파트를 작은 파트로 쪼갠다는 의미**
> 
> **[ 정의 ] : 하나의 Collection 의 데이터를 여러 개로 분산해서 저장한다**
> 
>> <strong> [ 배경 ] </strong>
>> 처리량이 많은 애플리케이션이나 매우 큰 데이터를 처리할 때는 하드웨어의 성능이 중요시 됩니다. 또한 높은 쿼리 속도는 디스크 드라이브의 CPU, RAM

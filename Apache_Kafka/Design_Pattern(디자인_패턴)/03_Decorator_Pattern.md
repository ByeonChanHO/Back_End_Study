# 3. Decorator Pattern(데코레이터 패턴)

    객체의 결합을 통해 추가적인 기능을 동적으로 유연하게 확장할 수 있게 해주는 패턴이다.

    즉, 기본 기능에 추가 할 수 있는 기능의 종류가 많은 경우에 각 추가 기능을 Decorator 클래스로 정의한 후
    
    필요한 Decorator 객체를 조합함으로써 추가 기능의 조합을 설계하는 방식이다.

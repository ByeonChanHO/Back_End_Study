# CRUD
> ## [ 정의 ]
>> CRUD 는 DB관련 업무를 하면서 많이 쓰이는 용어로 데이터 처리 기능인 Create(생성), Read(읽기), Update(갱신), Delete(삭제) 를 묶어서 일컫는 말
>>
